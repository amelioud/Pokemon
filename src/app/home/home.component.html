Our Application What is the objective of our application?

The focus of our application is on the theme of pokemons. This application works via an API that collects 250 Pok√©mon cards and displays them. The site will allow maps to be displayed according to pokemon type, as well as search for the desired pokemon. The pokemon cards being in English we decided to make the whole site in English.

Site features Some features we used

We use the Pokemon TCG API, this API aims to provide us with a database of all pokemon cards. It provides within these cards all the necessary characteristics such as the name of the pokemon, its type or even its attacks. This API offers different functionalities. It allows us to get all cards, get a specific card, get types and subtypes, and other useful features.
